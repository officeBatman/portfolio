Lately, I've been doing a lot of Python programming with NumPy.
One of the cool features of NumPy is that it allows you to create views of arrays.
For example, if I have a matrix and I want to modify a row, I can write this: matrix = np.zeros((3,3)) first_row = matrix[0, :] first_row += 1 This increments every element in the first row!
This idea can be extended to multiple dimensions too. matrix = np.zeros((3,3,3)) first_row = matrix[0, :, :] first_row += 1 Well maybe row is not the right term, but you get the idea.
